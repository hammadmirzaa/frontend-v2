# tenants

Super-admin tenant management pages for listing, creating, and inspecting tenants in the meichat platform.

## Files

| File | Description |
|------|-------------|
| `TenantCreatePage.jsx` | Form page for creating a new tenant with optional initial admin user and max-user seat configuration. |
| `TenantDetailPage.jsx` | Detail page for a single tenant showing usage, audit logs, configuration, seat-limit handling, and the embedded subscription console. |
| `TenantListPage.jsx` | Paginated, searchable list of tenants with quick links to create, configure, restore, and delete each tenant. |
